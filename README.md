# 🛡️ משחקי ממ"ד

אפליקציית משחקים לשימוש במרחב המוגן — עובדת במלואה ללא חיבור לאינטרנט.

---

## מה זה?

חוברת של 60 משחקים חברתיים לכל המשפחה, שניתן לשחק בהם בממ"ד בלי ציוד, בלי אינטרנט ובלי הכנה מוקדמת. המשחקים מתאימים לילדים ולמבוגרים, ומיועדים להרגיע ולהסיח את הדעת בזמן אירוע חירום.

---

## איך משתמשים?

1. הורידו את הקובץ `index.html`
2. פתחו אותו בכל דפדפן (Chrome, Safari, Firefox)
3. בחרו משחק מהרשימה — ותתחילו לשחק!

אין צורך בשרת, אין צורך בחיבור לרשת. הקובץ עובד לגמרי עצמאי.

---

## תוכן האפליקציה

**60 משחקים** מחולקים ל-5 קטגוריות:

| קטגוריה | תיאור | דוגמאות |
|---|---|---|
| 💬 מילים | משחקי שפה ומילים | אסוציאציות, שרשרת חרוזים, מילה אסורה |
| 🏃 תנועה | משחקים פיזיים | פנטומימה, עמידה על רגל אחת, ים יבשה |
| 📖 דמיון | סיפורים ויצירתיות | פסיכולוג הפוך, מכירה פומבית, כוכב שומר |
| ❤️ רגש | משחקים חברתיים ורגשיים | תכונה טובה, אמת על עצמי, טלפון שבור |
| 🏆 אתגר | תחרויות וניחושים | 20 השאלות, חם קר, המלך אמר |
